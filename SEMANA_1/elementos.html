INVESTIGACION DE TAGS

"!DOCTYPE html"

Se utiliza al momento de configurar una página web,con este sabremos en qué versión de HTML estamos trabajando, para que el navegador de texto pueda procesar la información de una forma correcta.

"html /html"

Representante la raíz de un documento HTML o XHTML.Funciona para poder darle una buena estructura y poder distribuir de buena manera el contenido de la página web,los demás elementos deben ser descendientes de este elemento.Con slash indica el cierre del documento HTML.

"head /head"

En este tag se encuentra todo el contenido de una página web, representando una colección de metadatos acerca del documento, incluyendo enlaces o definiciones, con la diagonal se indica el cierre de dicha etiqueta.

"title /title"

Esta etiqueta siempre va dentro de un head y se puede ver en las pestañas del navegador o en la barra de título. Define el título del documento, el cual se muestra en la barra de título del navegador o en las pestañas de página, se usa la diagonal para cerrar la etiqueta como se mencionó antes.

"body /body"

Se utiliza para poder poner dentro de el todo el contenido y diseño de una página web,es una etiqueta fundamental para el diseño de páginas web. Representa el contenido principal de un documento HTML. Solo hay un elemento body en un documento.

"h1 /h1"

Es una etiqueta que nos sirve para indicar algunas palabras clave que se le puede dar al contenido de una página web,implementan seis niveles de cabeceras de documentos; h1 es la de mayor y h6 es la de menor importancia.

"p /p"

Se utiliza para poder distribuir bien los párrafos de la página.
