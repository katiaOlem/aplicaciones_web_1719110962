"!DOCTYPE html"

Define que el documento está bajo el estándar de HTML 5

"html /html"

Representa la raíz de un documento HTML o XHTML. Todos los demás elementos deben ser descendientes de este elemento. con slash indica el cierre del documento HTML.

"head /head"

Representa una colección de metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo. Con la diagonal se indica el cierre de dicha etiqueta.

"title /title"

Define el título del documento, el cual se muestra en la barra de título del navegador o en las pestañas de página. Solamente puede contener texto y cualquier otra etiqueta contenida no será interpretada. Por otro lado, se usa la diagonal para cerrar la etiqueta como se mencionó antes.

"body /body"

Representa el contenido principal de un documento HTML. Solo hay un elemento body en un documento.

"h1 /h1"

Los elementos de cabecera implementan seis niveles de cabeceras de documentos; h1 es la de mayor y h6 es la de menor importancia. Un elemento de cabecera describe brevemente el tema de la sección que introduce.

"p /p"

Define una parte que debe mostrarse como un párrafo.
